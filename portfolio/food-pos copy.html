Сложный интерфейс Food Pos

Иван Быстрицкий
25 Августа, 2021

#react #redux #react-router #api #custom-ui-library

Разработал приложение для ресторанов. Создал собственную UI библиотеку компонентов под проект. Грамотно разбил на компоненты интерфейс. Научился разрабатывать сложные элементы интерфейса.

Какие функции были реализованы:

  • Красивый переход по страницам
  • Получение данных из мок апи
  • Фильтрация данных
  • Поиск
  • Добавление в карзину
  • Оформление заказа
  • Валидация форм
  • Разработка независимых ui компонентов
